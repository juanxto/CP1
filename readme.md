# CP01 - Mobile Application Development

Aplicação desenvolvida utilizano React Native trabalhando autentição e banco de dados da Firebase.
Esse projeto tem como objetivo desenvolver funcionalidades de autenticação básica e métodos CRUD padrão.

## Como executar o programa
````shell
git clone https://github.com/juanxto/CP1
cd CP1
npm install
Insira as credenciais no firebaseConfig.js
npx expo start --android (ou emulador equivalente)
````
